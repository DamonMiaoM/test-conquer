# Claude Code 指南

## 项目

**Test Conquer**：在线模拟考试平台，支持上传 PDF/DOCX 模拟题，提供 Web 端考试环境。

## 项目类型

开发 + 课程同步

## 核心功能

- 上传 PDF、DOCX 格式的模拟题文件
- 自动解析题目内容
- Web 端模拟考试环境（计时、答题、交卷）
- 成绩统计与错题回顾

## 项目结构

- **docs/** - 项目文档、需求说明
- **notes/** - 开发笔记、技术调研
- **src/** - 源代码
- **public/** - 静态资源

## 规则

- 文档使用中文，代码注释中英文均可
- 不自动提交，除非用户明确要求
- 敏感信息存 `.env.local`，不入 Git
- 项目开发与课程进度同步推进
